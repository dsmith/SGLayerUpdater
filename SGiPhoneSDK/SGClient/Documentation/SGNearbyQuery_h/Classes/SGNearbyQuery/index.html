SGNearbyQuery

A query object that defines common properties in order to provide all search filters for a nearby request.

Superclass: NSObject
Declared In: SGNearbyQuery.h

This property is not required in order to fulfill a successful query.

types

An array of record types that will help filter the search. 

@property (
    nonatomic,
    retain) NSArray* types;
Discussion

This property is not required in order to fulfill a successful query. If no types are specified then all types will be used to filter out the nearby request.
