SGAuthorization

Encapsulates HTTP request sent from SGLocationService with the proper authorization.

Extends Protocol: NSObject
Declared In: SGAuthorization.h

Discussion

The only class that implemenets this protocol is SGOAuth. The main uses of this protocol is to add authentication to an HTTP request. Currently, the only type of authentication that is supported by SimpleGeo is OAuth.
